# Security Runbook

Security operations focus on access control, auditability, consent, and launch gating.

## Controls in Repo

- MFA is required for privileged roles.
- Audit logs track access and sensitive actions.
- Citizen consent is recorded and checked before simulation access.
- Launch readiness includes security and privacy review gates.

## Response Flow

- Investigate audit anomalies.
- contain the affected workflow
- confirm whether partner or citizen data was touched
- document remediation in governance review cadence
