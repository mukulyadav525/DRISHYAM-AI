# DRISHYAM API Reference

The API is organized under `/api/v1`.

## Product Modules

- `/auth`, `/security`, `/privacy`
- `/detection`, `/honeypot`, `/voice`, `/forensic`, `/mule`, `/upi`, `/telecom`
- `/bharat`, `/notifications`, `/recovery`, `/pilot`, `/program-office`

## Control Plane

- `/program-office/national-scale` returns rollout, capacity, and escalation planning.
- `/program-office/business` returns pricing, billing, pipeline, and ROI surfaces.
- `/program-office/support` returns queues, manuals, tickets, and review cadence.
- `/program-office/documentation`, `/program-office/governance`, `/program-office/launch-readiness`, and `/program-office/continuous-improvement` cover post-MVP governance.
