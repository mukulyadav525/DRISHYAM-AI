# Dataset Documentation

The current repo is demo-safe and does not ship live training datasets. It documents the dataset types needed for production.

## Dataset Families

- call metadata and complaint history
- scam transcripts and honeypot exchanges
- payment identifiers, mule patterns, and recovery outcomes
- deepfake media evidence and annotation records

## Handling Rules

- keep raw PII out of public or demo artifacts
- track lineage, labeling quality, and partner approval before production use
