# IVR and USSD Operations

This guide covers multilingual alert and citizen support channels that do not require a smartphone.

## Current Coverage

- IVR confirmation and warning flows
- USSD and SMS content planning through the Bharat layer
- multilingual support scripting for pilot and scale phases

## Operator Guidance

- Always confirm language before asking the user to act.
- Use short instructions, explicit scam red flags, and recovery next steps.
- Escalate to live support when a victim reports recent financial loss.
