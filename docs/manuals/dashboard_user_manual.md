# Dashboard User Manual

The dashboard is the operator console for detection, alerts, pilot launch, national scale, business ops, support, and governance.

## Key Workspaces

- Command and agency pages drive operational response.
- Launch, national, business, ops, and governance pages manage post-MVP execution.
- Settings exposes MFA, audit, and privacy state.

## Daily Flow

- Verify MFA.
- Review command and alert posture.
- Work partner, support, and governance queues.
- Close the day by checking launch readiness and improvement backlog.
